PdmV/ValDb JSON API page

Posibilities:

release_info:

Takes: release_name

Returns: Release Full information

all_releases:

Takes:

Return: List of existing releases

add_releas:

Takes: release_name, catSubCat, relmon_url*

Return: Status of adding new release (Success or Failure)

Example: 

/add_release?release_name=apiTEST12&catSubCat=RData,PData

*: Not mandatory field
